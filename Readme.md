# Api Intial setup


- node init 
```js
npm init -y
```

- set typescript 
```js
npm i -D typescript ts-node nodemon @types/node
```

- after install ts config file setup
```js
npx tsc --init
```

- add dev script in `package.json`
```js
"dev":"nodemon server.ts"
```
